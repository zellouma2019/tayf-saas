## 🚀 نشر سريع على Vercel (5 دقائق)

### الخطوة 1: إنشاء قاعدة بيانات Turso مجانية (9 جيجا)

1. اذهب إلى [turso.tech](https://turso.tech) وسجّل حساب مجاني
2. من لوحة التحكم، أنشئ قاعدة بيانات جديدة باسم `tayf-db`
3. انسخ **رابط الاتصال** (Connection URL) — سيبدو هكذا:
   ```
   libsql://tayf-db-zellouma2019.turso.io
   ```
4. أنشئ **رمز مصادقة** (Auth Token) من صفحة الإعدادات

### الخطوة 2: النشر على Vercel

**الطريقة الأسهل — عبر GitHub:**
1. اضغط على زر "Deploy with Vercel" أعلاه
2. أو اذهب إلى [vercel.com/new](https://vercel.com/new) واختر المستودع `tayf-saas`
3. في صفحة المتغيرات البيئية (Environment Variables)، أضف:
   - `TURSO_DATABASE_URL` = الرابط من الخطوة 1
   - `TURSO_AUTH_TOKEN` = الرمز من الخطوة 1
4. اضغط "Deploy" وانتظر البناء (حوالي دقيقتين)

### الخطوة 3: تهيئة قاعدة البيانات

بمجرد فتح الموقع، قاعدة البيانات ستُهيّأ تلقائياً عند أول محاولة دخول.
كلمة مرور المدير الافتراضية: `Admin@2025`

## 🛠️ التقنيات

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript 5
- **Styling**: Tailwind CSS 4 + shadcn/ui
- **Database**: Prisma ORM + SQLite (محلي) / Turso (سحابي - 9GB مجاناً)
- **State**: Zustand + TanStack Query
- **Animation**: Framer Motion
- **Charts**: Recharts
- **PDF**: jsPDF

## 🏗️ التشغيل المحلي

```bash
# تثبيت المكتبات
bun install

# إعداد قاعدة البيانات
bun run db:push

# تشغيل الخادم
bun run dev

# بناء نسخة الإنتاج
bun run build && bun start
```
