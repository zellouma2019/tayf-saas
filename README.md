# طيف (Tayf) — منصة إدارة المطابع SaaS

> منصة متكاملة متعددة المستأجرين لإدارة المطابع ومكاتب الخدمات الطباعية في الوطن العربي

## ✨ المميزات الرئيسية

### للتجار (أصحاب المطابع)
- **لوحة تحكم متكاملة** — إدارة الطلبات، العملاء، المصاريف، والتحليلات
- **تتبع الطلبات** — لوحة كانبان لتتبع حالة كل طلب
- **طباعة إيصال حراري** — دعم طابعات الإيصالات 58mm
- **إدارة الزبائن** — قاعدة بيانات كاملة مع الإحصائيات
- **تحليلات متقدمة** — رسوم بيانية وتقارير مفصلة
- **قوالب نماذج رسمية** — إنشاء نماذج مخصصة
- **تصدير البيانات** — CSV/Excel
- **إشعارات ذكية** — تنبيهات الطلبات الجديدة

### للزبائن
- **طلب طباعة سهل** — واجهة بسيطة وسريعة
- **تحليل ذكي للملفات** — يكتشف نوع الملف والصفحات تلقائياً
- **تتبع الطلب** — متابعة حالة الطلب لحظة بلحظة
- **فاتورة PDF** — تنزيل فاتورة احترافية
- **مساعد ذكي (AI)** — دعم بالذكاء الاصطناعي والصوت
- **إعادة طلب سابق** — بضغطة واحدة
- **دعم RTL** — واجهة عربية كاملة

### للمنصة (المدير)
- **إدارة متعددة المتاجر** — لوحة تحكم مركزية
- **نظام ميزات مجانية/مدفوعة** — تحكم كامل بتفعيل الميزات
- **إحصائيات شاملة** — تقارير لكل المتاجر
- **إدارة الفريق** — صلاحيات متعددة

## 🌍 الدول والعملات المدعومة

يدعم المنصة 22 دولة عربية بالإضافة إلى عملات دولية:
- 🇩🇿 دينار جزائري (DZD) | 🇸🇦 ريال سعودي (SAR) | 🇪🇬 جنيه مصري (EGP)
- 🇦🇪 درهم إماراتي (AED) | 🇲🇦 درهم مغربي (MAD) | 🇹🇳 دينار تونسي (TND)
- 🇶🇦 ريال قطري (QAR) | 🇰🇼 دينار كويتي (KWD) | 🇯🇴 دينار أردني (JOD)
- 💵 دولار أمريكي (USD) | 💶 يورو (EUR) | 💷 جنيه إسترليني (GBP)
- والمزيد...

## 🛠️ التقنيات

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript 5
- **Styling**: Tailwind CSS 4 + shadcn/ui
- **Database**: Prisma ORM + SQLite
- **State**: Zustand + TanStack Query
- **Animation**: Framer Motion
- **Charts**: Recharts
- **PDF**: jsPDF
- **AI**: z-ai-web-dev-sdk (LLM, VLM, TTS, ASR)

## 🚀 التشغيل المحلي

```bash
# تثبيت المكتبات
bun install

# إعداد قاعدة البيانات
bun run db:push

# تشغيل الخادم
bun run dev

# بناء نسخة الإنتاج
bun run build && bun start
```
