# طيف (Tayf) — ملخص المشروع العام

## ما هو طيف؟
**طيف** هو منصة SaaS متكاملة لإدارة طلبات الطباعة، مصممة خصيصاً للسوق العربي (الجزائر، السعودية، الإمارات، تركيا...). يوفر لكل متجر طباعة (مكتبة خدمات، مطبعة، مركز تصوير) واجهة احترافية مستقلة يشاركها مع عملائه عبر رابط مباشر، مما يُغني عن تلقي الطلبات عبر واتساب أو الهاتف.

## البنية التقنية

| التقنية | الاستخدام |
|---------|-----------|
| **Next.js 16** (App Router) | الإطار الأساسي — SSR + Client Components |
| **TypeScript 5** | لغة التطوير |
| **Tailwind CSS 4 + shadcn/ui** | التصميم والواجهة |
| **Prisma ORM + SQLite (Turso)** | قاعدة البيانات |
| **Lucide Icons** | الأيقونات |
| **next-themes** | الوضع الداكن/الفاتح |
| **Zustand** | إدارة الحالة (العميل) |
| **Sonner** | الإشعارات (Toast) |
| **z-ai-web-dev-sdk** | مهارات الذكاء الاصطناعي (TTS, ASR, VLM, LLM, Image Gen) |

### البنية المعمارية
- **180 ملف TypeScript** في المشروع
- **42 نقطة نهاية API** (REST)
- **42 مكون واجهة** (Components)
- **8 قوالب لونية** للمتاجر
- **25 ميزة** (7 مجانية + 18 مدفوعة)
- نشر على **Vercel** (Serverless)
- بوابة **Caddy** للـ Gateway

## واجهة المستخدم

### للزبون (العميل)
- **شاشة ترحيب (Intro)** — قابلة للتخصيص بالكامل (ألوان، نصوص، مدة، إيموجي)
- **إنشاء طلب طباعة** — سحب وإفلات الملفات، تحليل ذكي تلقائي (صفحات، حجم، نوع)
- **6 أنواع خدمات**: مستندات، صور، تجليد، نسخ، بطاقات، ملصقات
- **خيارات التوصيل** — متعددة مع رسوم وأوقات تقديرية
- **تتبع الطلب** — بالرقم أو الرابط المباشر
- **سجل الطلبات** — مع إمكانية تكرار طلب سابق
- **زر واتساب عائم** — للتواصل المباشر
- **الوضع الداكن** — زر تبديل
- **دعم RTL** كامل
- **تذييل احترافي** — معلومات الاتصال، ساعات العمل، روابط

### للتاجر (صاحب المتجر)
- **لوحة تحكم كاملة** — تُصل بالرابط `?admin=1`
- **إدارة الطلبات** — عرض، فلترة، تغيير الحالة، تفاصيل كاملة
- **لوحة كانبان** — سحب وإفلات لتغيير حالات الطلبات
- **إحصائيات** — إجمالي الطلبات، الإيرادات، الربح، الأداء اليومي
- **تحليلات متقدمة** — رسوم بيانية، اتجاهات، أفضل العملاء
- **إدارة العملاء (CRM)** — بيانات العملاء، سجل الطلبات لكل عميل
- **إدارة المصاريف** — تتبع تكاليف التشغيل
- **إعدادات المتجر**:
  - معلومات المالك والاتصال
  - رفع الشعار (ضغط تلقائي)
  - اختيار القالب اللوني + لون رئيسي مخصص
  - تخصيص شاشة الترحيب
  - أسعار الخدمات وأقسامها
  - خيارات التوصيل
  - خصومات الكمية والوجهين
  - رسالة ترحيب مخصصة
  - حد أدنى للطلب
- **إعدادات متقدمة**:
  - تخصيص كامل للخدمات والأقسام والأسعار
  - إعدادات شاشة الترحيب (ألوان، نصوص، مدة، عناصر)
  - إعدادات التوصيل المتقدمة
  - الدولة والعملة
- **مشاركة الرابط** — نسخ، QR Code، واتساب
- **معاينة المتجر** — iframe مباشر داخل لوحة التحكم
- **تصدير Excel** — للطلبات
- **طباعة إيصالات حرارية** — تنسيق خاص للطابعة الحرارية
- **فواتير PDF** — إنشاء فواتير احترافية
- **إجراءات جماعية** — تغيير حالة أو حذف عدة طلبات
- **المساعد الذكي (AI)** — دردشة، تحليل الملفات، توليد صور
- **قوالب النماذج** — نماذج مخصصة لجمع بيانات العملاء

### للمشرف (Super Admin)
- **لوحة إدارة شاملة** — إدارة كل المتاجر
- **إنشاء متاجر جديدة**
- **إحصائيات عامة** — عدد المتاجر، الطلبات، الإيرادات
- **تحليلات يومية**
- **إدارة فريق العمل**
